# Ortese_Joelho
Desenvolvimento de órtese de joelho instrumentada
"O projeto de pesquisa aborda o desenvolvimento de uma órtese de joelho dotada de sensores capazes de levantar dados relativos aos movimentos da articulação e mecanismos para restringir e/ou corrigir os movimentos do joelho. Deste modo a órtese proposta terá dupla finalidade: diagnóstico de certas patologias associadas a esta articulação e tratamento ou correção dos movimentos realizados. A instrumentação embarcada será utilizada por uma interface homem-máquina, a ser desenvolvida e otimizada, para o diagnóstico das eventuais patologias e acompanhamento da evolução do tratamento."
